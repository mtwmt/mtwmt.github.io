input chexcbox

2017-06-23 | css

概念

利用 css before、after 的特性 去修改裡面的 input 樣式
在 before 裡畫出一個 ✔
使用矩形其中兩個長寬邊畫 border 線條
再利用 css3 rotate 屬性 旋轉 45 度 讓長寬兩個邊看起來像 ✔
再來 將其比例縮小到 scale 0 點下去後 再還原至原本大小 scale 1

after 畫下面底框，蓋住原本 checkbox 的樣式
