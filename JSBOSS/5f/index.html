這次的關卡比較大的問題點就應該就是在 CORS 的問題處理上
畢竟要用前端處理公開 api 這件事，本應由後經由後端處理，前端只要負責串接即可
但因為安全考量，API 端如果沒有開啟 CORS 的話，就變成只能看得到吃不到 RRRRR
