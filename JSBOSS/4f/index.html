時區清單

Timestamp

timestamp 一般被指為 Unix 時間戳，以 Unix 紀元開始算起 (西元1970年1月1日 00:00:00) 並且通常以秒為單位，在某些系統會採用這種時間格式，在各種語言也都支援這種時間戳格式轉換

如何取得timestamp

使用js將本地時間轉排為另一個時區

1秒 = 1000毫秒
1分鐘 = 60秒
1小時 = 3600秒

將1分鐘轉換為毫秒 = 60*1000 = 60000
1小時 = 60000*60 = 3600000
