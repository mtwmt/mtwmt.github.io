d3 range 屬於 d3-array 的一部份

源始碼如下

export default function (start, stop, step) {
  (start = +start),
    (stop = +stop),
    (step = (n = arguments.length) < 2 ? ((stop = start), (start = 0), 1) : n < 3 ? 1 : +step);

  var i = -1,
    n = Math.max(0, Math.ceil((stop - start) / step)) | 0,
    range = new Array(n);

  while (++i < n) {
    range[i] = start + i * step;
  }
  return range;
}

解析

型別轉換

(start = +start),
(stop = +stop),

拆解後如下

start = Number(start);
stop = Number(stop);

因為要做計算所以 將傳進來的值強制轉為數字型別

可參考 你懂 JavaScript 嗎？#8 強制轉型（Coercion）

三元運算子

step = (n = arguments.length) < 2 ? ((stop = start), (start = 0), 1) : n < 3 ? 1 : +step;

拆解後如下

if ((n = arguments.length) < 2) {
  // 如果傳進來的參數只有一個值
  stop = start;
  start = 0;
  step = 1;
} else {
  if (n < 3) {
    step = 1;
  } else {
    step = +step;
    // step = Number(step);
  }
}

迴圈

d3 這裡使用 while 迴圈

var i = -1,
  n = Math.max(0, Math.ceil((stop - start) / step)) | 0, //算出要跑幾次迴圈
  range = new Array(n); // 再賦予 n 個空陣列

while (++i < n) {
  range[i] = start + i * step;
}

來說說 ++i 與 i++ 的差別

++i

迴圈執行前 先 +1 如同上面 一開始宣告 i = -1; 所以在執行迴圈時 i 會先+1 然後再去跑迴圈 所以這個 while 會從 0 開始跑

i++ 後執行

迴圈執從後 才 +1 如用上述 這麼寫的話 會先從 -1 開始跑

for 與 while 的差別

重新認識 JavaScript: Day 09 流程判斷與迴圈 在這 kuro 大大的這篇文章最後面裡有提到 這兩者的差異性 這裡就不多說了 XD

使用函數

帶入起始值 與 結束點，從起始值開始，每個階段的累加值

d3.range(3, 70, 8);
// [ 3, 11, 19, 27, 35, 43, 51, 59, 67 ]

如果沒有給它累加的值，則每次就只會累加 1

d3.range(3, 10);
// [3, 4, 5, 6, 7, 8, 9]

如果只傳進來一個值

d3.range(5);
// [0, 1, 2, 3, 4]

range(start: number, stop: number = start, step: number = 1) {
  if( arguments.length < 2 ){
    start = 0;
  }
  const n = Math.max(0, Math.ceil((stop - start) / step));
  const range = new Array(n);
  for (let i = 0; i < n; i++) {
    range[i] = (start + i * step)
  }
  return range;
}
range(start: number, stop: number = start, step: number = 1, arr: Array<number> = [start]) {
  if (arguments.length < 2) {
    start = 0;
    arr = [0];
  }
  if (start + step >= stop) {
    return arr;
  } else {
    start = start + step;
    arr = [...arr, start]
    return this.range(start, stop, step, arr);
  }
}
