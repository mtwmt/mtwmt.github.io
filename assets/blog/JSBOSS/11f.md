## fetch

在取圖片的時後 使用`fetch`來取發現 `fetch` 在請求 api 資料時 沒有帶參數的功能，
必需寫在網址上

[照片搜尋參數](https://unsplash.com/documentation#search)

| 參數        | 功能                                   |
| ----------- | -------------------------------------- |
| w，h        | 用於調整照片的寬度和高度               |
| crop        | 用於對照片進行裁剪                     |
| fm          | 用於轉換圖像格式                       |
| auto=format | 用於根據用戶瀏覽器自動選擇最佳圖像格式 |
| q           | 用於在使用有損文件格式時更改壓縮質量   |
| fit         | 用於在指定尺寸內更改圖像的適合度       |
| dpr         | 用於調整圖像的設備像素比率             |
