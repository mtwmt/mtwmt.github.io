jQuery-extend

jqueryJavaScript

這篇應該算是踩雷過後的學習記錄

var a = { a: 1 };
a.aa = $.extend( a, { b:2 });

這樣寫的話 會造成 a 的無窮迴圈

a.aa = $.extend( {},a, { b:2 });
或
$.extend( true, a.aa ,a, { b:2 } );

才能正常的做物件深度合併
