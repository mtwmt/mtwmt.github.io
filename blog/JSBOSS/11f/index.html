LightBox 概念

首先我先將所有的資訊都放在 photo 的列表裡點了圖後 直接取得 li 裡的 data 資訊後 放到 lightbox 上有助後續若要修改圖片的話 只要改一個地方就好

fetch

在取圖片的時後 使用fetch來取發現 fetch 在請求 api 資料時 沒有帶參數的功能， 必需寫在網址上

照片搜尋參數

參數功能
w，h用於調整照片的寬度和高度
crop用於對照片進行裁剪
fm用於轉換圖像格式
auto=format用於根據用戶瀏覽器自動選擇最佳圖像格式
q用於在使用有損文件格式時更改壓縮質量
fit用於在指定尺寸內更改圖像的適合度
dpr用於調整圖像的設備像素比率
