接著繼續處理未完的專案

情境

這次的情境是，表單裡有包含著圖片上傳，機制是當表單需要 新增圖片時，需要先將圖檔上傳，得到後端給的圖片 id 後再把這個 id 塞到表單裡 更新圖片時，把原有的圖檔 id 包含新的圖檔一起傳送給後端， 刪除圖片時，就要把原有的圖片 id 打到後端 以上這些動作都要先處理完，接著再一起更新表單

所以流程會是，先傳送圖片，等圖片處理完後更新表單，再將表單送出。
