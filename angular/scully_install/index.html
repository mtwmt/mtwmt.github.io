卡關點

來分享下我在學習 sully 時 花比較多時間理解的地方

因為思考著是否將原本的 hexo 改用 scully
而原本的 hexo 網址前面沒有 blog 所以希望能在網址不要更動之下轉移

在網址對應這部份

例：

app-routing.module.ts

scully.scully.config.ts 裡的 routes 路徑 必須與 app-routing.module.ts 裡的 path 對應
